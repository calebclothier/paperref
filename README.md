# PaperRef

## Getting Started

### Installing uv

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

### Starting the backend server 

   ```bash
   cd paperref/backend
   uv run fastapi dev
   ```

### Starting the frontend web app 

   ```bash
   cd paperref/frontend/src
   uv run streamlit run Home.py
   ```
